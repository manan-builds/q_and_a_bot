Domain-Specific Q&A Bot (RAG-based)

A Retrieval-Augmented Generation (RAG) chatbot that answers questions from your own PDF documents (like syllabus, notes, or policies) and provides source citations.

Built using LangChain, ChromaDB, OpenAI, and Gradio.

Features:

Ask questions from your private PDFs  
Answers grounded in document context  
Source citations (PDF name + page)  
Simple confidence scoring  
Gradio UI  
Evaluation: With RAG vs Without RAG
